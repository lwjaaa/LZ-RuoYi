# Claude Code 高频极简命令速查（截图留存版）

## 一、会话基础

/reset 重置项目上下文
/clear 清空对话
/reload 刷新项目结构
/exit 退出

## 二、文件代码

/read 读取文件
/edit 编辑代码
/refactor 全局重构
/undo 撤销修改
/format 代码格式化

## 三、运行&排错

/run 执行命令
/test 运行测试
/diag 一键诊断报错
/search 全局搜索
/grep 正则检索

## 四、Git 操作

/git status 变更查看
/git diff 改动对比
/commit 生成提交

## 五、Superpowers 核心（AI 编程必用）

/superpowers:brainstorm 需求拆解
/superpowers:write-plan 输出开发方案
/superpowers:execute-plan 自动按计划开发
/superpowers:debug 系统化排错
/superpowers:review 代码自检优化

## 六、权限&插件

/skills list 查看已装技能
/plugin list 查看插件
/reload-plugins 重载插件
/allow-all 临时全开权限

## 七、我的常用

/resume 恢复会话，选中后，按两下EAS可以回到会话
/ide 连接idea vscode插件
