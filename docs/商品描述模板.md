1. 当DESCRIPTION、SIZE、MATERIAL有值的时候拼接一下代码段，其中DESCRIPTION、SIZE、MATERIAL对应的所在P标签要按需展示，比如SIZE没有内容的时候，就不展示`<p><span style="color: rgb(64, 64, 64);"><strong>Size: </strong>{{SIZE}}</span></p>`这行。

```html
<p style="display: flex; align-items: center; font-weight: bold; font-size: 18px;"><svg height="25" width="25"
    xmlns="http://www.w3.org/2000/svg" version="1.1" viewbox="0 0 1024 1024" class="icon">
    <path fill="#ffc439" d="M166.692 234.795h692.46l-343.068 593.65z"></path>
  </svg> DETAIL</p>
<p style="color: rgb(64, 64, 64);"><strong>Description: </strong><span id="my_description">{{DESCRIPTION}}</span></p>
<p style="color: rgb(64, 64, 64);"><strong>Size: </strong><span id="my_size">{{SIZE}}</span></p>
<p style="color: rgb(64, 64, 64);"><strong>Material: </strong><span id="my_material">{{MATERIAL}}</span></p>
```

2. 当NOTE有值的时候，展示：

```html
<p> </p>

  </svg> NOTE</p>
<div id="my_note" class="table-wrapper" uia-uid="379|1">{{NOTE}}</div>
```
