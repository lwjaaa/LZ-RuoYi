```text

│ ├── datasource/ # 数据源

│ │ ├── components/ # 公共组件
│ │ ├── store/ # Pinia 状态管理
│ │ ├── router/ # 路由配置
│ │ ├── utils/ # 工具函数
│ │ └── layout/ # 布局组件
│ └── vite/plugins/ # Vite 插件配置
├── ruoyi-ui/ # ⚠️ 已废弃（不要在此目录修改）
```
